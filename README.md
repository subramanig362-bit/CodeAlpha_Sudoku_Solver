#  Sudoku Solver (9X9)

This project is Task 3 of the CodeAlpha C++ Internship.  
It is a simple Sudoku Solver that uses the backtracking technique to complete a 9x9 Sudoku puzzle.

The user enters the puzzle with **0 representing empty cells**, and the program fills the entire Sudoku grid if a solution exists.

---

## Features
- Accepts a 9x9 Sudoku input
- Uses backtracking to solve the puzzle
- Checks row, column, and 3x3 box rules
- Prints solved Sudoku or shows message if no solution exists
- Beginner-friendly implementation
